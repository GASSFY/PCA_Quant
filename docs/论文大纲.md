# Task-Subspace Alignment as a Cross-Modal Signal for Channel-Wise Adaptive Quantization
# 任务子空间对齐作为跨模态信号用于通道级自适应量化

## 摘要
现有混合精度 PTQ 多依赖权重幅值、激活或 Hessian 类指标，但很少利用任务子空间几何来刻画通道敏感性。通过实验，我们观察到视觉模块与语言模块的激活子空间几何不同，但存在共享的**任务子空间对齐**信号。对此，我们提出了**Gate**，用通道权重在任务子空间上的相对投影比例做 bit allocation。Gate 在最终 mixed-precision 分配上优于复合 proxy，而 L1_gate_mul 更适合作为误差解释器，而不是最终排序器。

---

## 1 Introduction
### 0.1 研究背景
LLM / MLLM 推理成本高，混合精度 PTQ 是重要方向。现有通道级重要性通常关注：权重幅值、激活强度、二阶近似，但是这些指标没有显式利用任务相关输入方向。

### 0.2 核心问题
1. 不同模态模块（vision / language）的激活几何是否一致？
不同模态几何不一样，但都可以用任务子空间来分析。
2. 是否存在一个跨模态共享、并且适合通道级 bit allocation 的敏感性指标？
Gate 是一个跨视觉/语言模块都有效的通道敏感性信号。L1_gate_mul 相关性更强，但 gate-only 最终量化更优。这个差异对应**误差拟合**与**排序分配**两个目标的不一致。

### 0.3 贡献点
1. **PCA 实验揭示模态异质性**
我们通过 PCA 子空间分析定量揭示了多模态模型内部的模态异质性：视觉模块呈现低秩集中型激活几何，而语言模块呈现高秩分散型激活几何。

这会进一步说明，单纯靠统一的幅值启发式不够，因为：
- 在视觉侧，少数主方向特别重要；
- 在语言侧，重要方向更分散；
- 所以通道重要性应该由**相对于该模态、该层自己的任务子空间的对齐程度**来决定。

2. **提出 Gate**
基于任务激活主要位于低维子空间这一观察，我们提出 Gate，用通道权重与任务子空间的相对对齐程度刻画通道敏感性，并将其用于通道级 mixed-precision bit allocation。

PCA 告诉我们“任务常用哪些方向”，Gate 则衡量“每个通道有多少能量放在这些方向上”。

3. **发现“误差相关性最强”与“最终 bit allocation 最优”并不等价**
L1_gate_mul 更像误差解释器，而 gate-only 更适合作为最终排序信号。

## 2 Motivation
### 2.1 任务激活子空间
给定层输入激活 a，统计协方差 $\sum_{a}$
做 PCA，取前 k 个主成分形成 $U_{k}$
这些方向代表任务/数据分布下最常用的输入变化方向。

### 2.2 跨模态几何差异
对视觉层，第一主成分占比较高、特征值衰减快，能量更集中。
对语言层，特征值衰减更缓、能量更分散。

如果在较小 k 时 E(k) 很高，说明能量集中在少数方向，是低秩主导；否则更分散。

### 2.3 为什么子空间几何和量化有关
如果任务相关激活集中在低维子空间，那通道敏感性应更多由**是否对准该子空间**决定；因而应该用几何对齐，而不是单纯幅值，来刻画通道重要性。

如果某通道权重主要对准任务常用方向，那么量化它更可能伤害真实输出。因此通道重要性不应只看幅值，还应看对任务子空间的对齐程度。

## 3 Method
提出并介绍 Gate

### 3.1 问题定义
给定某层权重矩阵 W，对每个输出通道 $W_{i}$ 分配不同 bit。
目标是在固定预算下最小化量化带来的任务损失。

### 3.2 Gate 的定义
PCA 子空间分析 → 发现任务子空间结构 → 用权重对子空间的相对对齐来定义通道重要性。

方法核心：
$$g_{i}=\frac{\left\|P_{k}w_{i}\right\|}{\left\|w_{i}\right\|+\epsilon}$$
其中，$P_{k}=U_{k}U_{k}^{T}$

- $w_{i}$：第 i 个输出通道的权重向量
- $P_{k}$：任务激活子空间的投影矩阵，通常由 PCA 主成分 U 构造
- 分子 $\left\|P_{k}w_{i}\right\|$：这个通道投影到任务子空间后的能量
- 分母 $\left\|w_{i}\right\|$：这个通道自身总能量

所以，Gate 本质上是在问：这个通道的总能量里，有多少比例落在**任务真正会用到的子空间**里。

落进去的比例和误差幅度是否有关？如果一个数值为10000的通道有50%的能量落进去了，那么可能会比落入60%能量、数值为1的通道更加重要。

### 3.3 从 Gate 到 bit allocation
算法流程：
1. 收集层输入激活
2. 计算协方差并做 PCA 得到 U
3. 计算每个通道的 Gate score
4. 按 Gate 排序
5. 分配高 bit/低 bit（或多级 bit）

## 4 实验
### 因子分解实验（消融实验）
我们发现，与量化误差相关性最强的指标并不一定对应最优的混合精度比特分配结果。具体而言，L1_gate_mul 在 LLM 与 ViT 上都表现出更强的 Pearson 相关性，说明其更适合作为量化误差幅值的解释变量；然而，在固定高精度通道预算下，gate-only consistently yields lower reconstruction error，说明其更适合作为最终通道排序信号。

该现象表明，通道级 mixed-precision PTQ 的核心并不只是误差回归，而是预算约束下的 top-k 选择，因此**最佳误差 proxy**与**最佳 allocation signal**并不等价。

两者的区别：
- **Gate-only**：纯几何信号，回答“这个通道和任务子空间到底有多对齐？”
- **L1_gate_mul**：几何信号+幅值信号，Importance = L1 × Gate，先算 L1 能量，再算 Gate，最后做乘法，强调“只要其中一项低，通道就不重要”

这也解释了实验现象：
- L1_gate_mul 像“误差解释器”
- gate-only 更像“最终排序器”
因为前者更贴近误差幅值，后者更贴近谁该被保护。

因子分解实验设计本身就是为了验证这一点。
